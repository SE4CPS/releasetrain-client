Communicate software releases with external stakeholders manually via a URL or automated via cURL or a Slack App Channel.

URL

Use the query string parameter (https://releasetrain.io/?q=query) to share software releases of components of interests to users or to display them on a dashboard. The URL gets automatically refreshed every four hours.


  • releasetrain.io/?q=browser //safari, chrome, firefox browsers
  • releasetrain.io/?q=mongo,express,angular,node //MEAN stack
  • releasetrain.io/?q=linux,apache,mysql,php //LAMP stack
  • releasetrain.io/?q=elasticsearch,logstach,kibana //ELK stack

cURL

Add this cURL command to your to your deployment build and publish releases via releasetrain.io. 

curl 
	-H 'Content-Type: application/json' https://releasetrain.io/api/v 
	-d '{
			"versionNumber": (String) '<MAJOR.MINOR.PATCH.BUILD>',
			"versionProductBrand": (String) e.g., 'Microsoft',
			"versionProductName": (String) e.g., 'Windows 10',
			"versionProductType": (Array) e.g., ['operating system'],
			"versionProductLicense": (String) e.g., 'commercial',
			"versionReleaseChannel" (String) e.g., 'alpha'/ 'beta' / 'prod',
			"versionReleaseNotes": (URL String) e.g., '<URL>',
			"versionReleaseDate":e.g., (Date String) '<YYYYMMDD>',
			"versionReleaseComments": (String) 'Release notes comments/summary',
			"versionSearchTags": (Array) e.g., ['operating system'],
			"versionId": (String) <versionProductName><versionNumber><versionReleaseDate>
		}'

Slack

Does your team use slack for communicating devops topics? Then install the slack releasetrain_bot app to get timely software release update notifications directly in slack. Enjoy!

  1. Add slack app releasetraion_bot to your slack account
  2. Add slack channel #release to your slack account
  3. Add slack user @releasetrain_bot to channel #release
